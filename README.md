# **REACT + VITE**

## Proyecto de React JS - Montero Rodrigo

### Este proyecto se basa en un ecommerce de venta de artículos Apple. Todo lo que figura en él es ficticio. Es un proyecto desarrollado para el curso de ReactJS de Coderhouse. 

## Funcionamiento de la página:

### Al iniciar la página, vemos el home de la misma, a través del cual podemos navegar hacia la tienda. En la tienda, obtenemos los productos a través de Firebase, donde se aloja en el Firestore la colección con los diferentes artículos y también una colección donde se van guardando las órdenes que se generan. Al finalizar la compra, se obtiene el ID de dicha orden para demostrar el buen funcionamiento de la página. 

### También utilizamos las rutas, donde a medida que vamos navegando en el sitio, la misma va cambiando dependiendo de dónde nos encontremos. Tenemos una ruta para el home, otra para la tienda, desde ahí podemos navegar a las diferentes categorías o incluso a cada uno de los productos obteniendo el detalle de los mismos. En cada card de productos vemos un botón de añadir, el mismo nos añade el producto al carrito, donde podemos a su vez aumentar o disminuir la cantidad del mismo. Una vez en el carrito, al finalizar la compra, debemos completar nuestros datos para poder enviarla y así el proceso de la página finaliza.
