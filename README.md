# **REACT + VITE**

## Proyecto de React JS - Montero Rodrigo

### Este proyecto se basa en un ecommerce de venta de articulos Apple. Todo lo que figura en él es ficticio. Es un proyecto desarrollado para el curso de ReactJS de coderhouse. 

## Funcionamiento de la pagina:

### Al iniciar la pagina vemos el home de la misma, a traves de la cual podemos navegar hacia la tienda. 

### En la tienda, obtenemos los productos a traves de firebase, en donde se aloja en el firestore la coleccion con los diferentes articulos y tambien una coleccion donde se van guardando las ordenes que se generan. Al finalizar la compra se obtiene el ID de dicha orden para demostrar el buen funcionamiento de la pagina. 

### Tambien utilizamos las routes, donde a medida que vamos navegando en el sitio la misma va cambiando dependiendo donde nos encontremos. Tenemos una route para el home, otra para la tienda, de ahi podemos navegar a las diferentes categorias o incluso a cada uno de los productos obteniendo el detalle de los mismos. En cada cart de productos vemos un boton de añadir, el mismo nos añade el producto al carrito en donde podemos a su vez aumentar o disminuir la cantidad del mismo. 

### Una vez en el carrito al finalizar la compra debemos completar nuestros datos para poder enviar la misma y asi el proceso de la pagina finaliza.
